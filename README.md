# Symmetric-File-Encryption-System
Secure File Encryption & Decryption System is a Linux-based file security tool that encrypts and decrypts files using a password-based XOR encryption technique. The program allows users to securely convert normal files into encrypted form and restore them using the same password.

The project uses Linux file handling system calls to read, write, and manage files securely.

Technologies Used
C++
Linux System Calls
File Handling
XOR Encryption Algorithm

Purpose

The purpose of this project is to demonstrate basic data security concepts by implementing password-based file encryption and decryption.
